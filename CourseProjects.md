# Course Projects

## ITAI 2373 — Natural Language Processing

This course focused on applying Natural Language Processing (NLP) techniques to solve real-world text analysis problems using Python, machine learning, and transformer-based models.

### Featured Project 1 — Midterm: NewsBot Foundation

The midterm project focused on building the foundation of a NewsBot Intelligence System for analyzing and classifying news articles.

Main components included:

* Text preprocessing and cleaning
* Sentiment Analysis
* Named Entity Recognition (NER)
* Syntactic Analysis using spaCy
* Machine Learning Classification

The classification model achieved approximately 0.78 accuracy, demonstrating effective article classification and sentiment analysis.

This project established the technical base for the final project.

---

### Featured Project 2 — Final: NewsBot Intelligence System 2.0

The final project expanded the midterm NewsBot into a more advanced and production-ready NLP platform.

Advanced features included:

* Topic Modeling (LDA)
* Text Summarization
* Semantic Search
* Multilingual Intelligence
* Conversational Interface
* Improved System Architecture

The system transformed raw news content into structured, multilingual, and actionable insights for faster decision-making and business intelligence.

This project demonstrated the evolution from a foundational NLP system into a complete intelligent platform.
